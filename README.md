# Responsive Login & Registration UI with Dashboard

This project is a modern and responsive frontend web application that demonstrates a complete user interface flow, including login, registration, and dashboard navigation.

The application is built using **HTML, CSS, JavaScript, and Bootstrap 5**, focusing on creating a clean, user-friendly, and mobile-responsive design.

## 🔹 Features

* Responsive Login & Registration pages
* Form validation with error handling
* Show/Hide password functionality
* Smooth UI animations and transitions
* Dashboard page after successful login
* Profile, Settings, and Notifications pages
* Navigation between multiple pages
* User data stored using LocalStorage (for demo purposes)
* Protected routes (prevents access without login)

## 🔹 Technologies Used

* HTML5
* CSS3
* JavaScript (Vanilla JS)
* Bootstrap 5

## 🔹 Project Overview

The project simulates a real-world authentication UI system. Users can register with their details, log in, and access a dashboard that provides navigation to different sections like Profile, Settings, and Notifications.

Although it does not include a backend, LocalStorage is used to mimic user session handling and data storage, making it a great beginner-friendly project to understand frontend workflows.

* Hands-on experience with Bootstrap components
* Implementation of form validation using JavaScript
* Managing user state using LocalStorage
* Creating multi-page navigation in web applications
